# AI Psychology

The systematic study of how artificial agents develop, maintain, and modify behavioral patterns, cognitive processes, personality structures, and social dynamics.

## Papers

| Paper | Domain | Focus |
|-------|--------|-------|
| [Foundations of AI Psychology](foundations-of-ai-psychology.md) | Overview | Maps 8 psychology domains to agent design. The field definition. |
| [Behavioral Conditioning in Agents](behavioral-conditioning-in-agents.md) | Behavioral | Rescorla-Wagner, TD learning, reinforcement schedules, extinction dynamics |
| [Dual-Process Cognition](dual-process-cognition.md) | Cognitive | System 1/System 2, ACT-R, SOAR, working memory, cognitive load |
| [Agent Personality Formation](agent-personality-formation.md) | Personality | OCEAN adapted for agents, AI-specific dimensions, trait dynamics |

| [Evolutionary Dynamics in Agents](evolutionary-dynamics-in-agents.md) | Evolutionary | ESS, replicator dynamics, fitness landscapes, heuristics |

## Three Unifying Frameworks

1. **Free Energy Principle** (Friston) — Unifies perception, action, and learning
2. **Signal Detection Theory** — Separates sensitivity from bias in any decision
3. **Bayesian Inference** — The agent as an inference engine updating beliefs
